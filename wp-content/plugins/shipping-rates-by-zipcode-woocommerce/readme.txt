You want to manage shipping rates by zipcode? This plugin is totally for you to manage postcodes and shipping fee. You can easily add postcodes and shipping cost accordingly. Checkout will calculate shipping fee by added ZipCode automatically.

== Description ==

**Shipping Rates by PostCode** is the most advanced shipping plugin for WooCommerce stores allowing you to calculate the shipping costs based on Zip Code, Post Code or Postal code. It will allow you to manage ZipCode list and shipping cost.<br>
If you want to allow just selected postcodes to ship order. it's best fit for you. 
<br><br>


**How it work?**
* goto woocommerce > settings > shipping
* select Flat Shipping Rate by ZipCode
* Enable plugin and add ZipCode list 
* save settings

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the WooCommerce -> Settings -> shipping -> Flat Shipping Rate by ZipCode
1. Complete given form and submit
1. it's ready to use
